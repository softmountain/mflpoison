# GAN 实验自动化

## 🚀 快速开始（推荐使用自动化脚本）

### 方式 1：使用自动化脚本（推荐）

```bash
# 单个实验运行
bash run_gan_experiment.sh --exp_name "MoMOnly_0.5" --gan_mom_weight 0.5

# 批量运行多个实验
bash run_gan_experiments_batch.sh

# 快速测试脚本是否正常工作
bash test_automation.sh

# 用假数据训练模型
bash run_gan_experiment.sh \
  --mode fake_train \
  --exp_name "FakeTrain_CLS_LOW" \
  --gan_checkpoint results/local_gan/ckpt_200_CLS_LOW.pt \
  --fake_epochs 50

# 用假数据进行标签翻转攻击
bash run_gan_experiment.sh \
  --mode fake_attack \
  --exp_name "FakeAttack_3to7" \
  --gan_checkpoint results/local_gan/ckpt_200_CLS_LOW.pt \
  --attack_src_label 3 \
  --attack_dst_label 7 \
  --attack_prob 0.7
```

**优势：**
- ✅ 自动保存完整日志到 `results/logs/`
- ✅ 自动运行训练和评估
- ✅ 智能命名文件
- ✅ 支持批量实验

详细使用方法见 [GAN_EXPERIMENT_README.md](./GAN_EXPERIMENT_README.md)

## 📝 手动命令（原始方式）

### gan 基准测试
```bash
python train_local_gan.py \
  --model_path results/local_training/best_model.pt\
  --gan_epochs 200 \
  --gan_lr_g 2e-4 \
  --gan_lr_d 1e-4 \
  --gan_rf_weight 2.0 \
  --gan_aux_weight 1.0 \
  --gan_cls_weight 0.1 \
  --gan_joint_weight 0.05 \
  --gan_fm_weight 0.0 \
  --gan_mom_weight 0.0 \
  --gan_audio_out_max 10.0 \
  --gan_audio_scale_max 5.0 \
  --gan_video_out_max 20.0 \
  --gan_video_scale_max 8.0 \
  --batch_size 32 \
  --audio_feat mfcc \
  --video_feat mobilenet_v2
```

#### 先测“输出约束”组件（因为它直接对应 audio 爆方差）
```
python train_local_gan.py \
  --model_path results/local_training/best_model.pt\
  --gan_epochs 200 \
  --gan_lr_g 2e-4 \
  --gan_lr_d 1e-4 \
  --gan_rf_weight 2.0 \
  --gan_aux_weight 1.0 \
  --gan_cls_weight 0.1 \
  --gan_joint_weight 0.05 \
  --gan_fm_weight 0.0 \
  --gan_mom_weight 0.0 \
  --gan_audio_out_max 1.0 \
  --gan_audio_scale_max 0.3 \
  --gan_video_out_max 20.0 \
  --gan_video_scale_max 8.0 \
  --batch_size 32 \
  --audio_feat mfcc \
  --video_feat mobilenet_v2
```
**之后所有实验（MoM/FM、cls/joint 强度）都在“输出约束 ON”的前提下做对照，否则 audio 爆方差会把其它组件的效果淹没，结论不干净。**
